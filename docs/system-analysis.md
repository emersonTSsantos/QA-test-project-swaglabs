# Análise de Sistemas - SwagLabs

## Visão Geral da Aplicação

O SwagLabs é uma aplicação de demonstração de e-commerce usada para testes práticos.

O sistema permite que os usuários façam login, naveguem pelos produtos, adicionem itens ao carrinho e concluam o processo de finalização da compra.

## Principais Funcionalidades

1. Login do Usuário
2. Listagem de Produtos
3. Adicionar Produto ao Carrinho
4. Remover Produto do Carrinho
5. Carrinho de Compras
6. Processo de Finalização da Compra
7. Confirmação do Pedido

## Riscos de Teste

Possíveis riscos identificados durante a análise:

- Falhas na validação do login
- Preços incorretos dos produtos
- Falhas de ortografia na Descrição do Produto
- Erros no Nome do Produto
- Carrinho com atualização incorreta
- Problemas na validação da finalização da compra
