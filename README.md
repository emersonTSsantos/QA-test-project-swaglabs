# Projeto de Teste Manual de QA - SwagLabs

Este projeto simula as atividades diárias de um Analista de QA trabalhando em uma equipe ágil.

O objetivo deste projeto é demonstrar o processo de garantia da qualidade em um cenário real, incluindo análise de requisitos, planejamento de testes, elaboração de testes e execução.

O sistema em teste é o aplicativo de demonstração da Swag Labs:
https://www.saucedemo.com/

Este aplicativo simula uma plataforma de e-commerce e é comumente usado para testes práticos.

## Objetivos do Projeto

- Análise de requisitos
- Criação de histórias de usuário
- Planejamento de testes
- Documentação de testes
- Execução de testes manuais
- Elaboração de casos de teste
- Cenários BDD
- Gerenciamento do ciclo de testes

## Ferramentas e Conceitos Utilizados

- Agile / Scrum
- Jira (simulação)
- Confluence (simulação de documentação)
- Teste manual
- Teste de caixa preta
- BDD (Gherkin)
- Mapeamento mental para elaboração de testes

## Escopo do Teste

As seguintes funcionalidades serão analisadas e testadas:

- Login do usuário
- Listagem de produtos
- Adicionar produto ao carrinho
- Remover produto do carrinho
- Processo de finalização da compra
